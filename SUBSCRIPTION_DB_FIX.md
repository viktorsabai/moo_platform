# Подписки: если в ЛК «Ошибка» и нет планов у юзера

**Сделай один раз после деплоя:**

1. Залогинься в админку (владелец ресторана).
2. Открой в браузере ссылку (подставь свой домен):
   ```
   https://ТВОЙ-ДОМЕН.vercel.app/api/admin/db/apply-subscription-columns
   ```
3. Должен открыться ответ `{"ok":true,...}`. Колонки в БД добавлены.
4. Обнови страницу ЛК подписок — ошибка пропадёт, планы появятся.

**Если при создании подписки юзером видишь ошибку**  
`The column SubscriptionPlanTemplate.minDishesPerDelivery does not exist` — значит миграция не применена. Открой ту же ссылку выше (apply-subscription-columns) один раз; после этого создание подписок заработает. До применения миграции код подставляет значения по умолчанию, но надёжнее добавить колонки в БД.
