## 📊 Если нужна база данных (опционально)

### Быстрое решение - используй SQLite (легче, чем PostgreSQL)

1. **Измени `prisma/schema.prisma`:**
   В начале файла замени `provider = "postgresql"` на `provider = "sqlite"`

2. **Создай файл `.env`** в корне проекта:
   ```env
   DATABASE_URL="file:./dev.db"
   NEXTAUTH_SECRET="любая-случайная-строка-здесь"
   NEXTAUTH_URL="http://localhost:3000"
   # Telegram (нужно для авторизации через mini app)
   BOT_TOKEN="123456:ABCDEF..."
   ```

3. **Запусти миграции:**
   ```bash
   npx prisma migrate dev --name init
   ```

4. **Перезапусти приложение:**
   ```bash
   npm run dev
   ```
