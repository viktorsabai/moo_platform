# UFO Delivery - Веб-приложение доставки еды для ресторана и магазина

Современное веб-приложение для доставки еды и товаров магазина с прозрачной операционкой для покупателя и продавца.

## MVP Scope (текущий запуск)

- Доступны два каталога: `меню ресторана` и `магазин`.
- Главная страница с баннерами остаётся точкой входа для пользователя.
- Подписки на рационы в MVP отключены как продукт, вместо них показывается умная заглушка с CTA "Хочу подписку".
- CTA на подписки создаёт заявку и отправляет уведомление в Telegram команде заведения (OWNER/ADMIN/STAFF).
- Доставка выполняется силами продавца, с настраиваемыми зонами и условиями в ЛК.
- Статусы заказа прозрачны и синхронизированы между клиентом, админкой и Telegram-уведомлениями.

## 🗄️ База данных

Схема базы данных описана в `prisma/schema.prisma`. Основные модели:

- **User** - Пользователи
- **Account/Session** - Аутентификация (NextAuth)
- **Category** - Категории блюд
- **Dish** - Блюда
- **Cart/CartItem** - Корзина
- **Order/OrderItem** - Заказы
- **Subscription/SubscriptionItem** - Подписки
- **SubscriptionDelivery** - Доставки по подписке
- **Address** - Адреса доставки

## 🔑 Основные функции

### 1. Меню и каталог
- Просмотр блюд по категориям
- Фильтрация и поиск
- Детальная информация о блюдах

### 2. Корзина
- Добавление/удаление блюд
- Изменение количества
- Расчет итоговой суммы

### 3. Заказы
- Оформление одноразовых заказов
- Отслеживание статуса заказа
- История заказов

### 4. Подписки (MVP)
- Подписки не продаются в первой версии
- Пользователь видит умную заглушку с CTA "Хочу подписку"
- Заявка на интерес к подписке уходит команде заведения в Telegram

### 5. Личный кабинет
- Профиль пользователя
- Управление адресами доставки
- История заказов и подписок

### 6. Платежи
- Интеграция с Stripe
- Безопасная обработка платежей
- Поддержка разных методов оплаты

### 7. Доставка (MVP)
- Настройка зон доставки в ЛК продавца
- Для зоны задаются стоимость, минимальный чек и окно доставки
- На checkout пользователь видит прозрачный расчёт по своей зоне

## 🛠️ Установка и запуск

### Требования
- Node.js 18+
- PostgreSQL 14+
- npm или yarn

### Шаги установки

1. **Клонирование репозитория**
   ```bash
   git clone <repository-url>
   cd UFO_Delivery
   ```

2. **Установка зависимостей**
   ```bash
   npm install
   ```

3. **Настройка переменных окружения**
   ```bash
   cp .env.example .env
   # Отредактируйте .env файл, указав свои настройки
   ```

4. **Настройка базы данных**
   ```bash
   # Создайте базу данных PostgreSQL
   # Обновите DATABASE_URL в .env файле
   
   # Запустите миграции
   npx prisma migrate dev
   
   # (Опционально) Заполните базу тестовыми данными
   npx prisma db seed
   ```

5. **Запуск в режиме разработки**
   ```bash
   npm run dev
   ```

   Приложение будет доступно по адресу: http://localhost:3000

6. **Сборка для продакшена**
   ```bash
   npm run build
   npm start
   ```

## 📝 API Endpoints

### Аутентификация
- `POST /api/auth/signin` - Вход
- `POST /api/auth/signup` - Регистрация
- `GET /api/auth/session` - Получить сессию

### Меню
- `GET /api/dishes` - Получить список блюд
- `GET /api/dishes/:id` - Получить блюдо по ID
- `GET /api/categories` - Получить категории

### Заказы
- `GET /api/orders` - Получить заказы пользователя
- `GET /api/orders/:id` - Получить заказ по ID
- `POST /api/orders` - Создать заказ
- `PATCH /api/orders/:id` - Обновить заказ

### Подписки
- `POST /api/subscription-requests` - Отправить заявку "Хочу подписку" (MVP)

### Платежи
- `POST /api/payment/intent` - Создать платежный интент
- `POST /api/payment/webhook` - Webhook для обработки платежей
